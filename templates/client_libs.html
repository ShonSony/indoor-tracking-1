The API is designed to allow easy, intuitive access from any language that can make HTTP requests (i.e., any language!)

However, programming a mobile device to collect RSSI and/or magnetic field data can be difficult and prone to error.

Thankfully, we have developed an Android library that you can use to ease the pain.

While we don't have any other platforms yet (for example, iOS or JavaScript), we are planning to implement a similar library in the future.

If you are interested in contributing such a library, please let us know!
