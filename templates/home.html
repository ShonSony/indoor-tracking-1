Fingerprints Collector helps you record, organise and access wifi RSSIs and magnetic fields.

As well as providing a data corpus to use for research, you can use this website to track devices that can record wifi RSSI using our REST API.

You can get an Android library to facilitate collecting data and using our API here.

Everyone, including you, can see and use the data available. If you upload data, you (and only you) will be able to edit or delete it later.

If you do use our data for a scientific publication, please remember to cite
